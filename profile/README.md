# Computer Sound Portfolio
This Github Organization includes assignments from CS516 Computer Sound and Music.

# About Me
I am a Graduate student at Portland State University studying Computer Science.

I have some experience playing with many different kind of instruments.  Recently, I've grown an interest in learning about sound system in our Church.  

We use Presonus Studio 64 to mix many different microphones and instruments.  Many of the lectures covered seems very familiar such as topics like high pass filter, eq, cutting frequencies.

Through this course, I hope to use my software developing skills to replicate tools that I've used before and also learn more about sound.
